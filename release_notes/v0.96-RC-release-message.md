# v0.96-RC — Frame-condition correction & lattice birefringence prediction

A targeted correctness pass on the frame-condition derivation, prompted
by a SymPy verification, replaces a previously asserted isotropy with a
sharp, falsifiable directional prediction.

## The headline change

The earlier text claimed Σ_RGB v_i v_j^T = 3·δ_ij. Direct computation
shows the actual frame matrix is anisotropic:

```
M = [[3, -1, 1],
     [-1, 3, 1],
     [1, 1, 3]]      eigenvalues {4, 4, 1}
```

It's a prolate ellipsoid in **k**-space with optical axis along
**V₁ + V₂ + V₃ = (1, 1, −1)**. SO(3) rotational invariance is recovered
only on O_h-averaged observables; the residual operator-level anisotropy
is a closed-form **lattice birefringence**.

This is a strengthening of the framework, not a weakening — the paper
now makes a sharp, direction-resolved prediction (P7) where it
previously asserted a generic isotropy that wasn't actually derived.

## Reproducibility

Five lines of SymPy reproduce the central numeric claim:

```python
import sympy as sp
V1, V2, V3 = sp.Matrix([1,1,1]), sp.Matrix([1,-1,-1]), sp.Matrix([-1,1,-1])
M = V1*V1.T + V2*V2.T + V3*V3.T
print(M.tolist(), M.eigenvals())
# [[3, -1, 1], [-1, 3, 1], [1, 1, 3]]   {4: 2, 1: 1}
```
