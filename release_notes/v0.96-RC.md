## Summary

A targeted correctness pass on the frame-condition derivation, surfaced by a SymPy verification, revealed that the paper's previous claim
$\sum_\text{RGB}\mathbf{v}_i\mathbf{v}_j^T = 3\delta_{ij}$
was false. The actual frame matrix has off-diagonal entries $\pm 1$ and eigenvalues $\{4, 4, 1\}$. The corrected derivation recovers $\mathrm{SO}(3)$ rotational invariance only on $O_h$-averaged observables; the residual operator-level anisotropy is a lattice birefringence with closed-form coefficients and an optical axis along $\mathbf{V}_1 + \mathbf{V}_2 + \mathbf{V}_3 = (1,1,-1)$.

This is **a strengthening, not a weakening**: the framework now makes a sharp, falsifiable directional prediction (vacuum birefringence) where it previously asserted a generic isotropy that was not actually derived. The same direction-resolved analysis that tests the new prediction also tests the source of the Zitterbewegung rate $\omega$ (a new dual-hypothesis paragraph in §6).

## Reproducibility

The build is unchanged: `cd paper && bash build.sh` (POSIX) or `paper\build.cmd` (Windows) produces `main.pdf` from a clean checkout. The numerical claims about the frame matrix and birefringence eigenvalues are verifiable in 5 lines of SymPy:

```python
import sympy as sp
V1 = sp.Matrix([1,  1,  1])
V2 = sp.Matrix([1, -1, -1])
V3 = sp.Matrix([-1, 1, -1])
M = V1*V1.T + V2*V2.T + V3*V3.T
print(M.tolist(), M.eigenvals())
```

This produces `[[3, -1, 1], [-1, 3, 1], [1, 1, 3]]` and `{4: 2, 1: 1}` (eigenvalue 4 with multiplicity 2, eigenvalue 1 with multiplicity 1). The full derivation chain — from this matrix to the spinor propagator's effective $M_\text{eff}$ and the optical axis $(1,1,-1)$ — is reproduced in `src/utilities/frame_matrix_visualization.py` and described in `notes/frame_condition_isotropy_memo.md`.
