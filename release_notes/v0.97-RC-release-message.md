recoil-driven Zitterbewegung) inertial-mass anisotropies all share a
single optical axis. **Five independent observation channels** are

1. Kinematic photon dispersion (P7 GRB time-of-flight)
2. Vacuum gauge-sector birefringence (GRB / CMB polarization rotation)
3. Gravitational birefringence (LIGO/Virgo polarization-mode timing)

5. Anomalous-dispersion anisotropy (atomic-clock spectroscopy)

modulation (atomic clocks; fast handle) and redshift-resolved binning

recoil-driven Zitterbewegung reading, dynamical embedding is the
empirical signature of an externally-driven (vs. self-contained)

## Reproducibility

Five lines of SymPy reproduce Appendix B's central numeric claim:

```python
import sympy as sp
V1, V2, V3 = sp.Matrix([1,1,1]), sp.Matrix([1,-1,-1]), sp.Matrix([-1,1,-1])
F = sp.Matrix(3, 3, lambda i,j: sp.Symbol(f"F{i}{j}", real=True)*(1 if i<j else (-1 if i>j else 0)))
proj = lambda Va, Vb: sum(Va[i]*Vb[j]*F[i,j] for i in range(3) for j in range(3))
Q = sum(sp.expand(proj(V, W)**2) for V, W in [(V1, V2), (V1, V3), (V2, V3)])
# Extracting the quadratic form coefficient yields Q with eigenvalues {4, 4, 16}.
```
