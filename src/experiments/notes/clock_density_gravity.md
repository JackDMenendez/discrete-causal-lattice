# clock_density_gravity.md

## Core Claim
Gravitational time dilation is not geometric curvature.
It is scheduler load: more clocks near a mass = more processing overhead
per external tick = apparent time slowing from outside.

## The Information Spectrum
- Distant star: sparse information received locally -> few clocks -> baseline tick rate
- Near a planet: moderate clock density -> measurable time dilation
- Near a black hole: clock density -> saturation -> processor deadlock -> event horizon
- Black hole interior: infinite queue depth -> time stops from outside perspective

## Hawking Radiation Connection
Saturated processor occasionally drops a clock from the queue.
Information leaks stochastically. Maps to Gambler's Ruin from v1.0.
Black hole information paradox: information is queued, not destroyed.

## Holographic Principle Connection
Clocks live on the boundary (where information exchange occurs).
Interior clock count proportional to boundary area, not volume.
This is *why* the holographic bound has the form it does.
